# Investy App

This is an investment portfolio app used to track all your investments in one place.


### Installing

Getting Started:

```
$ bundle install
```

Run server:
```
$ shotgun
```
Create database:
```
$ rake db:migrate
```
Start app with data:
```
$ rake db:seed
```


## Built With

* Sinatra
* ActiveRecord
* Postgres
* Boostrap
